# Vasoconstriction Data

Data on vasoconstriction (lung constriction) response, from Table 13.2.

## Format

A data frame with 39 rows and 2 variables:

- volume:

  Volume measurement

- constriction:

  Logical: TRUE if constriction occurred, FALSE otherwise
